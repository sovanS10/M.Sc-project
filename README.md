# Brain Tumor Detection – M.Sc Project

## Description
This project is developed as part of my M.Sc coursework.
It focuses on detecting brain tumors using machine learning / deep learning techniques.

## Project Structure
- Training/        → Model training code
- Testing/         → Testing and evaluation
- saved_models/    → Trained models
- brain_tumor_app.py → Main application file

## How to Run
1. Install required libraries
2. Run:
   streamlit run brain_tumor_app.py
